# Getting Started
### Fork and Clone the repo

```
git clone hhttps://github.com/hhhrrrttt222111/developer-portfolio.git
cd developer-portfolio
```
### Install packages from the root directory
```
npm install
```
#### or
```
yarn add
```
### Start the development server
```
npm start
```
#### or
```
yarn start
```
